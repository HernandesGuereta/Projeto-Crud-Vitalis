# 🩸 Vitalis

O Vitalis é uma plataforma inteligente de gestão de doadores e automação de notificações para hemocentros, criada com o objetivo de auxiliar no controle de estoque de sangue e incentivar a doação recorrente.

O sistema conecta hemocentros e doadores através de alertas automáticos, pré-triagem digital e gerenciamento de campanhas de doação, promovendo maior agilidade e eficiência no processo de captação de sangue.

---

# 📌 Objetivo

O projeto busca solucionar problemas relacionados à baixa disponibilidade de bolsas de sangue, permitindo que hemocentros identifiquem rapidamente doadores compatíveis e realizem notificações automáticas em situações de escassez.

Além disso, o Vitalis fornece ferramentas para:
- Controle de estoque sanguíneo
- Gestão de doadores
- Pré-triagem digital
- Histórico de doações
- Dashboard analítico
- Alertas inteligentes

---

# 🚀 Funcionalidades

## 👤 Doador
- Cadastro e login seguro
- Visualização do histórico de doações
- Controle automático do período de carência
- Recebimento de alertas de escassez
- Pré-triagem digital
- Consulta de hemocentros

## 🏥 Hemocentro
- Cadastro de unidades
- Controle de estoque sanguíneo
- Atualização de entrada e saída de bolsas
- Configuração de limites mínimos
- Confirmação de doações
- Dashboard de estoque
- Gestão de questionários de triagem

---

# 🧠 Regras de Negócio

- Controle automático de carência:
  - 60 dias para homens
  - 90 dias para mulheres
- Notificações apenas para doadores aptos
- Controle de estoque sem valores negativos
- Compatibilidade sanguínea automática
- Registro e rastreabilidade de operações críticas

---

# 🔒 Requisitos Não Funcionais

- Conformidade com a LGPD
- Criptografia de dados sensíveis
- Sistema responsivo
- Backup automático
- Alta disponibilidade
- Escalabilidade para campanhas de doação
