# 🩸 Projeto CRUD Vitalis

Sistema CRUD desenvolvido para o projeto Vitalis, com foco no gerenciamento de movimentações de bolsas de sangue em hemocentros.

## 📌 Funcionalidades

- Cadastrar movimentações
- Listar movimentações
- Editar registros
- Excluir registros
- Controle de entrada e saída de bolsas

## 🛠️ Tecnologias

- React
- Node.js
- Express
- MySQL

## 🎯 Objetivo

Simular o controle de estoque sanguíneo através de operações CRUD relacionadas às movimentações de bolsas de sangue.

## 📚 Projeto acadêmico

Projeto desenvolvido para fins educacionais com base no sistema Vitalis.
